HTML - HyperText Markup Language

O HTML é uma linguagem de marcação, utilizada na construção de sites. Suas marcas são conhecidas como "tags" e através dessas tags definimos a semântica do que será exibido, proporcionando uma melhor organização e definição do conteúdo.

Principais Tags:

Clique na tag para mais detalhes

  • Header - < h1 > até < h6 >
  • Paragraph - < p >
  • Mark - < mark >
  • Small - < small >
  • Itálico - < i >
  • Sublinhado - < u >
  • Negrito - < strong >
  • Listar ordenadas - < ol >
  • Listas desordenadas - < ul >
  • Elementos de lista - < li >
  • Âncoras - < a >
  • Linha horizontal - < hr >
  • Subscrito - < sub >
  • Sobrescrito - < sup >
  • Citação - < blockquote >
  • Fonte - < font >
  • Tachado (deletado) - < del >
  • Abreviação - < abbr >

Listar ordenadas

A tag < ol > é utilizada quando se deseja escrever uma lista de itens ordenados, que respeitam uma sequência. Dentro de uma lista sempre devemos utilizar a tag < li > para informar quais serão os itens dessa lista. As listas seguem um padrão básico:

< ol >
< li > item de lista < li >
< li > item de lista < li >
< li > item de lista < li >
< /ol >

Exemplo de Lista Ordenada:

  1. Arroz
  2. Feijão
  3. Carne
Voltar

Listas desordenadas

As listas desordenadas seguem o mesmo padrão das listas ordenadas, porém não ordenam os itens.

Exemplo de Lista Desordenada:

  • Arroz
  • Feijão
  • Carne

Âncoras

A tag < a > é utilizada para inserir âncoras no texto, ou seja, vincular demais documentos HTML, links externos, links internos. Sua estrutura básica é a seguinte:

< a href="#" > Texto < /a >

Onde "href" é a tag onde colocamos a URL do site, ou o link interno que queremos referenciar.
